🍽️ Cunto Macaan Restaurant Website

Project Overview

Cunto Macaan is a responsive restaurant website developed as part of the Web Application Development Project. The website showcases a modern restaurant offering authentic Somali cuisine while providing visitors with information about the restaurant, menu, chefs, special offers, customer testimonials, and table reservations.

The website was built using HTML, CSS, Bootstrap, and JavaScript to create a responsive and user-friendly interface.

⸻

Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* Visual Studio Code
* Git & GitHub

⸻

Website Features

* Responsive Bootstrap navigation bar
* Attractive home page
* About Us page
* Restaurant menu with:
    * Appetizers
    * Main Courses
    * Desserts
    * Drinks
* Gallery page
* Meet Our Chefs page
* Special Offers page
* Reservation form
* Customer Testimonials page
* Contact page
* Responsive layout for desktop, tablet, and mobile devices

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open index.html in your preferred web browser.
4. Navigate through the website using the responsive navigation bar.
